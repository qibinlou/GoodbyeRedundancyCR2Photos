GoodbyeRedundancyCR2Photos
==========================

A python script for photographers to delete CR2 photos mapping with the JPG photos which have been deleted. 


##Usage

You are recommended to place your photos as follows:
.
├── IMG_8284.JPG
├── IMG_8287.JPG
├── IMG_8288.JPG
├── IMG_8289.JPG
├── cr2
│   ├── IMG_8284.CR2
│   └── IMG_8286.CR2
